# Завдання 1. Ієрархія каталогів Linux

### 1. Перейти до кореневого каталогу та показати його вміст

``` bash
cd /
ls -la
```

Приклад результату: bin boot dev etc home lib media mnt opt proc root
run sbin srv sys tmp usr var

------------------------------------------------------------------------

### 2. Перейти до каталогу /etc та показати його вміст

``` bash
cd /etc
ls
```

Приклад результату: apt cron.d group hostname hosts passwd shadow ssh

------------------------------------------------------------------------

### 3. Перейти до каталогу /home та показати список користувачів

``` bash
cd /home
ls
```

Приклад результату: student

# Завдання 2. Файли, каталоги та посилання

### 1. Створити новий каталог у домашньому каталозі

``` bash
cd ~
mkdir lab2
```

### 2. Перейти у створений каталог

``` bash
cd lab2
```

### 3. Створити файл

``` bash
touch file.txt
```

### 4. Скопіювати файл з новим ім'ям

``` bash
cp file.txt file_copy.txt
```

### 5. Перейменувати копію

``` bash
mv file_copy.txt file_renamed.txt
```

### 6. Створити жорстке посилання

``` bash
ln file.txt file_hardlink.txt
```

### 7. Створити символічне посилання

``` bash
ln -s file.txt file_symlink.txt
```

### 8. Знайти файл за ім'ям

``` bash
find ~ -name "file.txt"
```

Приклад результату: /home/student/lab2/file.txt

------------------------------------------------------------------------

# Завдання 3. Права доступу

### 1. Переглянути права доступу файлу

``` bash
ls -l file.txt
```

Приклад: -rw-r--r-- 1 student student 0 Apr 20 12:00 file.txt

### 2. Надати файлу права тільки на читання

``` bash
chmod 444 file.txt
```

### 3. Надати власнику право на запис

``` bash
chmod u+w file.txt
```

### 4. Переглянути значення umask

``` bash
umask
```

Приклад: 0022

### 5. Встановити нове значення umask

``` bash
umask 022
```

------------------------------------------------------------------------

# Завдання 4. Користувачі

### 1. Створити нового користувача

``` bash
sudo useradd -m trainee
```

### 2. Встановити пароль

``` bash
sudo passwd trainee
```

### 3. Додати користувача до групи sudo

``` bash
sudo usermod -aG sudo trainee
```

### 4. Перевірити що користувач існує

``` bash
cat /etc/passwd | grep trainee
```

Приклад результату: trainee:x:1001:1001::/home/trainee:/bin/bash
